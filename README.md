# FEA-Handbook

An interactive, single-page reference document for Finite Element Analysis workflows in ANSYS Workbench — built entirely with vanilla HTML, CSS, and JavaScript.

[https://FEA-Handbook](https://vsrwt.github.io/FEA-Handbook/)

## Overview

This handbook consolidates 23 FEA workflow sections into a structured, visually navigable reference — covering everything from static structural and modal analysis to rotor dynamics, fatigue, and topology optimization.

**Live Demo →** [View on GitHub Pages](https://your-username.github.io/your-repo-name)

## Features

- Scroll-progress indicator and sticky navigation bar
- Section-by-section animated reveal on scroll
- Color-coded analysis type cards (Static / Dynamic / Thermal / Special)
- Visual flow maps for every workflow
- Formula callouts (Euler's buckling, frequency conversion)
- Contact type and meshing comparison tables
- Zero dependencies — no frameworks, no build tools

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Fonts: [Syne](https://fonts.google.com/specimen/Syne) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts
- Intersection Observer API for scroll animations
